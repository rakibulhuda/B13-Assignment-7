# 🌿 KeenKeeper — Keep Your Friendships Alive

> Your personal shelf of meaningful connections. Browse, tend, and nurture the relationships that matter most.

---

## 📖 About

KeenKeeper is a friendship management web app that helps you stay intentional about the people who matter most in your life. Set contact goals, track interactions, and never let an important relationship go cold again.

---

---

## ✨ Key Features

### 1. 📇 Friend Tracking Dashboard
View all your friends in a clean 4-column grid with their photo, days since last contact, relationship tags (WORK, FAMILY, HOBBY, TRAVEL), and a color-coded status badge — **Overdue**, **Almost Due**, or **On-Track** — so you always know who needs attention at a glance.

### 2. ⚡ Quick Check-In & Timeline
From any friend's detail page, log a **Call**, **Text**, or **Video** interaction in one click. Every interaction is instantly added to a global Timeline page where you can filter by interaction type and see your full relationship history.

### 3. 📊 Friendship Analytics
A live donut chart on the Stats page breaks down all your interactions by type — Call, Text, and Video — giving you a visual snapshot of how you're staying in touch with the people around you.

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/keenkeeper.git

# Navigate into the project
cd keenkeeper

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
